### Bundled widgets

### Keymap
* _ModKey + r_ :               Run application
* _ModKey + shift + r_ :       Run command on terminal
* _ModKey + q_ :               Quite application

#### Volume controls
* _ModKey + F10_ :             Mute toggle  
* _ModKey + F11_ :             Decrease volume
* _ModKey + F12_ :             Increase volume
